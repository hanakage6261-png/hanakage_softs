# 番号付きフォルダ直下の `_README.md` 書式モデル

このファイルは、`momonGA_systems` 直下の番号付きフォルダに置く  
`00_momonGA_master_README.md` のような説明書ファイルの書式見本です。

## 書き方のルール

- 1行目は、そのフォルダ自体の説明を書く見出しにする
- 最初の段落で、そのフォルダが何のためのフォルダかを書く
- その下に `## 収納物` を置く
- `## 収納物` の中では、フォルダ直下にあるファイルやサブフォルダを相対リンクで書く
- 各項目は「リンク」「説明」「区切り線」の順で書く
- `__pycache__` のような自動生成物は、通常は説明対象に含めない

## 書式見本

```md
# 00_momonGA_master フォルダについて

このフォルダは、momonGA_systems 全体の運用基準点となるファイルを置くためのフォルダです。

## 収納物

- [momonGA_registry.py](./momonGA_registry.py)

  システム全体の共通参照先を一元管理するファイルです。

---

- [config](./config/)

  全体設定用の JSON などを置くサブフォルダです。

---

- [docs](./docs/)

  設計書、説明書、書式見本を置くサブフォルダです。
```
